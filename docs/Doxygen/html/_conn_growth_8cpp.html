When using a MATRIX_TYPE = “complete” the amount of used memory increases by another order of magnitude. Once enough memory is used no more memory can be allocated and a “bsd_alloc” will be thrown. The following members of the connection constructor consume equally vast amounts of memory as the simulation sizes grow:

Detailed Description

Header file for XmlGrowthRecorder.h.

/

The XmlGrowthRecorder provides a mechanism for recording neuron's layout, spikes history, and compile history information on xml file:

  1. neuron's locations, and type map,
  2. individual neuron's spike rate in epochs,
  3. network wide burstiness index data in 1s bins,
  4. network wide spike count in 10ms bins,
  5. individual neuron's radius history of every epoch.

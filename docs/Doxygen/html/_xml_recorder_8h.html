Detailed Description

An implementation for recording spikes history on xml file.

/

The XmlRecorder provides a mechanism for recording neuron's layout, spikes history, and compile history information on xml file:

  1. neuron's locations, and type map,
  2. individual neuron's spike rate in epochs,
  3. network wide burstiness index data in 1s bins,
  4. network wide spike count in 10ms bins.
